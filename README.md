# std::unordered_map Python wrapper
A python wrapper class for the std::unordered_map class (c++). 

**Note:**
 - All methods that make use of iterators are excluded. 
 - Only the string type is supported.
 
## How to use
Build the C++ library:

```
$ make
```

Then run the demo code:

```
$ python3 main.py
Hello World
```
